# Color Reference

Spectre.Console provides 291 named colors.

## Usage

```csharp
// In markup
AnsiConsole.MarkupLine("[red]Error[/]");

// In styles
new Style(foreground: Color.Blue);
```

## Common Colors

| Color | Hex |
|-------|-----|
| Black | #000000 |
| White | #FFFFFF |
| Red | #FF0000 |
| Green | #00FF00 |
| Blue | #0000FF |
| Yellow | #FFFF00 |
| Cyan | #00FFFF |
| Magenta | #FF00FF |

## Greyscale

Black, Gray0-Gray100, White
