# Markup Reference

Complete reference for inline markup syntax.

## Colors

```csharp
[red]Text[/]        // Foreground
[on blue]Text[/]    // Background
[green on yellow]Text[/]
```

## Styles

```csharp
[bold]Bold[/]
[italic]Italic[/]
[underline]Underline[/]
[strikethrough]Strike[/]
[dim]Dim[/]
```

## Combine

```csharp
[bold red on white]Bold Red on White[/]
```

## Links

```csharp
[link=https://example.com]Click here[/]
```

## Escaping

```csharp
[[escape brackets]]  // Output: [escape brackets]
```

## Hex Colors

```csharp
[#FF0000]Red[/#FF0000]
[#00FF00 bg]Green background[/#]
```

See: Color Reference
