# Table Border Reference

All table border styles available.

## Available Borders

| Style | Description |
|-------|-------------|
| None | No border |
| Minimal | Minimal lines |
| MinimalHeavyHead | Heavy header |
| MinimalHeavySep | Heavy separator |
| MinimalLight | Light only |
| Simple | Simple |
| SimpleHeavy | Simple with heavy |
| SimpleDouble | Double lines |
| Rounded | Rounded |
| Heavy | Heavy |
| HeavyEdge | Heavy edges |
| HeavyRound | Heavy rounded |
| Double | Double |
| DoubleEdge | Double edges |

## Usage

```csharp
table.Border(TableBorder.Rounded);
table.Border(TableBorder.Double);
```
