# Emoji Reference

All emoji shortcodes supported.

## Usage

```csharp
AnsiConsole.MarkupLine("Hello :smile:!");
AnsiConsole.MarkupLine("Success :white_check_mark:");
```

## Common Emojis

| Shortcode | Emoji |
|----------|------|
| :smile: | 😊 |
| :thumbsup: | 👍 |
| :fire: | 🔥 |
| :rocket: | 🚀 |
| :check_mark: | ✅ |
